## Техническая архитектура

### Стек технологий

| Компонент | Технология |
|-----------|------------|
| **Framework** | Next.js 16.2.0 (App Router) |
| **UI Library** | React 19 |
| **Язык** | TypeScript 5 |
| **Стилизация** | TailwindCSS v4 |
| **UI компоненты** | Radix UI primitives |

---

## Быстрый старт

### Требования

- Node.js 20+
- npm 10+

### Установка

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Доступ: [http://localhost:3000](http://localhost:3000)

### Production-сборка

```bash
npm run build
npm start
```

## Функциональные роли

### Гость
| Страница | Описание |
|----------|----------|
| `/` | Лендинг платформы |
| `/apply` | Заявка на регистрацию психолога |

### Психолог
| Страница | Описание |
|----------|----------|
| `/login` | Авторизация |
| `/dashboard` | Личный кабинет |
| `/surveys` | Управление опросниками |
| `/builder` | Конструктор методик |
| `/results` | Результаты тестирований |
| `/reports` | Генерация отчётов |
| `/subscription` | Управление подпиской |
| `/profile` | Профиль пользователя |

### Администратор
| Страница | Описание |
|----------|----------|
| `/admin/(auth)/login` | Вход в панель |
| `/admin/applications` | Заявки на регистрацию психологов |
| `/admin/applications-request` | Запросы на рассмотрение заявок |
| `/admin/psychologists` | Управление психологами |
| `/admin/subscriptions` | Управление подписками |

### Клиент
| Страница | Описание |
|----------|----------|
| `/client/[surveyId]` | Прохождение теста по ссылке |
| `/p/[token]` | Публичное прохождение опросника |
